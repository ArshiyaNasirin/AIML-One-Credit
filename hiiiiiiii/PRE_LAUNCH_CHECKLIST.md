## 🆘 TROUBLESHOOTING QUICK REFERENCE

| Issue | Solution |
|-------|----------|
| Images not showing | Check image URL, re-upload image |
| GitHub stats blank | Verify GitHub username is exact |
| Styling looks different | Hard refresh (Ctrl+Shift+R) |
| Links don't work | Check URLs include https:// |
| Mobile menu broken | Check JS file is loaded |
| Animations jerky | Check browser supports CSS3 |
| Page slow to load | Compress images, minimize CSS |
| Fonts not right | Check Google Fonts loaded |
| Colors different | Clear browser cache |
| GitHub streak shows 0 | Check GitHub username correct |
